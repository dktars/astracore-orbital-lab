# 🛰 AstraCore Orbital Lab

A 2D orbital mechanics simulator built to explore spacecraft motion, gravitational systems, and mission simulation concepts.

## Overview

AstraCore Orbital Lab is the first core project in the AstraCore ecosystem.
It focuses on building a clean and expandable orbital simulation platform that can grow into mission rehearsal, flight software testing, and secure space systems visualization.

## Goals

* simulate gravity-driven orbital motion
* visualize bodies and orbital paths
* support reusable mission scenarios
* provide a clean architecture for future AstraCore projects

## Planned Features

* 2D body simulation
* orbital trails
* predefined scenarios
* simulation controls
* body information display
* modular simulation architecture

## Tech Stack
